# Manual de Usuario — Tempodoro v1.0

# 1. Introducción

Tempodoro es una aplicación móvil diseñada para ayudar a los usuarios a organizar sus tareas mediante la técnica Pomodoro, mejorando la concentración y la productividad.

La aplicación permite crear tareas, administrar sesiones de estudio, controlar descansos y visualizar estadísticas del tiempo dedicado al trabajo.

---

# 2. Requisitos

Para utilizar Tempodoro se requiere:

- Dispositivo Android.
- Android 7.0 (API 24) o superior.
- Espacio disponible para instalar la aplicación.

---

# 3. Instalación

1. Descargar el archivo APK.
2. Instalar la aplicación en el dispositivo Android.
3. Si el sistema lo solicita, habilitar la instalación desde fuentes desconocidas.
4. Abrir Tempodoro.

---

# 4. Inicio de sesión

Al iniciar la aplicación se muestra la pantalla de autenticación.

El usuario puede:

- Iniciar sesión con una cuenta existente.
- Registrarse como nuevo usuario.

---

# 5. Pantalla principal

La pantalla principal muestra la lista de tareas registradas.

Desde esta pantalla es posible:

- Crear una nueva tarea.
- Editar una tarea.
- Iniciar una sesión Pomodoro.
- Consultar estadísticas.

---

# 6. Crear una tarea

Para registrar una nueva tarea:

1. Presionar el botón **Nueva Tarea**.
2. Completar los campos:
   - Título.
   - Descripción.
   - Categoría.
   - Número de Pomodoros estimados.
3. Guardar la información.

La tarea aparecerá automáticamente en la lista principal.

---

# 7. Editar una tarea

Seleccionar la tarea y presionar el botón **Editar**.

El usuario puede modificar la información registrada y guardar los cambios.

---

# 8. Eliminar una tarea

Desde la lista principal seleccionar la opción de eliminar.

La tarea será removida de la base de datos.

---

# 9. Utilizar el temporizador Pomodoro

Para iniciar una sesión:

1. Seleccionar una tarea.
2. Presionar el botón **Pomodoro**.
3. Presionar **Iniciar**.

Durante la sesión también es posible:

- Pausar el temporizador.
- Reiniciarlo.
- Configurar la duración del estudio y descanso.
- Saltar entre estudio y descanso.

Al finalizar una sesión se mostrará una notificación y el progreso de la tarea será actualizado automáticamente.

---

# 10. Consultar estadísticas

Desde el menú principal seleccionar **Estadísticas**.

La aplicación mostrará:

- Total de tareas.
- Tareas completadas.
- Pomodoros realizados.
- Pomodoros pendientes.
- Tiempo total de estudio.

---

# 11. Solución de problemas

## No aparecen las estadísticas

Completar correctamente una sesión Pomodoro para que el progreso sea almacenado automáticamente.

## No llegan las notificaciones

Verificar que la aplicación tenga permisos para enviar notificaciones.

---

# 12. Cierre de sesión

Para salir de la aplicación basta con cerrar la aplicación o regresar a la pantalla de inicio de sesión si la funcionalidad está disponible.

---
